## 如何使用

- 按 Alt + E 打开面板
- 输入 Latex
- 按下回车（或点击 OK 按钮）
- 程序会自动将图片复制到剪贴板中（如果是在聊天窗口中打开的，则会自动粘贴进去）

## 快捷键和指令

- Alt + E - 打开主面板
- 上下箭头 - 查看历史记录


- //info - 打开关于页面
- //help - 查看帮助
- //exit - 退出程序
